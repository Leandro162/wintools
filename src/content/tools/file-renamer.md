你有没有遇到过这种情况：相机拍出来的照片叫 `IMG_0823.JPG`，工作文件散落一堆没有规律，想整理却要一个一个手动改，改到手软还没改完。这个工具就是为了解决这个问题而做的。

## 工具能做什么

- ✓ 批量重命名，一次处理几百个文件
- ✓ 自定义前缀 / 后缀（如 `工作_001`、`旅行_20260514`）
- ✓ 按修改日期、创建日期自动排序编号
- ✓ 支持正则表达式替换（高级用法）
- ✓ 免安装，下载即用，双击 `.exe` 打开
- ✓ 无广告、不联网、不收集任何数据

> 💡 操作前工具会自动备份原文件名列表，万一改错了可以一键还原，不用担心改坏。

## 使用教程

### 第一步：把文件拖进工具窗口

打开工具后，直接把文件夹或多个文件从资源管理器拖进工具的文件列表区域。支持混合拖入不同格式的文件，拖进来后左侧会显示原文件名预览。

![拖入文件后的界面截图](/images/file-renamer-step1.png)

### 第二步：设置命名规则

在右侧面板选择命名方式。最常用的是「前缀 + 序号」，比如输入前缀 `旅行_`，序号从 `001` 开始，文件就会变成 旅行_001.jpg、旅行_002.jpg……右侧实时预览新文件名，确认无误再执行。

![命名规则设置面板](/images/file-renamer-step2.png)

### 第三步：点击「开始改名」

确认预览没问题后，点击底部绿色的「开始改名」按钮。处理完成后会弹出成功提示，显示改名数量。如需撤销，点击「还原」按钮即可恢复原文件名。

## 常见问题

**支持哪些文件格式？**

不限格式，任何文件都可以改名，包括图片、视频、文档、压缩包等。工具只修改文件名，不会改动文件内容。

**Windows 7 能用吗？**

当前版本需要 Windows 10 或 11（64位）。Windows 7 因系统组件限制暂不支持，后续版本可能会兼容。

> ⚠️ 改名前请确保文件没有被其他程序打开，否则可能提示权限不足。关闭相关程序后重试即可。
