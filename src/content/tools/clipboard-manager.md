你有没有遇到过这种情况：先复制了一段文字，还没粘贴就不小心又复制了别的，前面那条就永远找不回来了。这个工具就是专门解决这个问题的。

## 工具能做什么

- ✓ 自动记录最近 50 条复制内容（文字 + 图片）
- ✓ 快捷键 `Alt + V` 随时唤出历史列表
- ✓ 点击任意一条即可重新粘贴
- ✓ 支持搜索历史记录
- ✓ 开机自启，后台静默运行，不占系统资源
- ✓ 内容只存本地，不上云，不联网

## 使用教程

### 第一步：下载后直接运行

无需安装，双击 `.exe` 即可运行。首次运行会询问是否开机自启，建议选「是」，这样每次开机都不用手动打开。

### 第二步：正常复制，工具自动记录

安装后正常使用电脑，每次按 `Ctrl+C` 复制的内容都会自动保存到历史记录里，不需要做任何额外操作。

### 第三步：需要时按快捷键调出

按 `Alt + V`，屏幕中央会弹出历史记录列表，显示最近复制的内容。用鼠标点击或方向键选择，回车确认，内容就会粘贴到当前光标位置。

## 常见问题

**会不会记录密码等敏感内容？**

会记录所有复制内容，包括从密码管理器复制的密码。如果不想记录某类内容，可以在设置里配置「排除特定应用」，例如排除 KeePass、1Password 等。

**历史记录保存在哪里？**

保存在本地 `%AppData%\ClipHistory\` 文件夹里，纯本地存储，不会上传到任何地方。
